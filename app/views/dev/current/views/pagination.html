Pagination rules

If there is one page

Do not show pagination, it may be helpful to still show a count.

If there are 5 or fewer pages

Show all numbers with no separator dots.

{{hmctsPagination({ results: { from: 1, to: 50, count: 250 }, next: { text: 'Next', href: '#' }, items: [{ text: '1', href: '#', selected: true }, { text: '2', href: '#' }, { text: '3', href: '#' }, { text: '4', href: '#' }, { text: '5', href: '#' }] })}}
{{hmctsPagination({ results: { from: 101, to: 150, count: 250 }, previous: { text: 'Previous', href: '#' }, next: { text: 'Next', href: '#' }, items: [{ text: '1', href: '#' }, { text: '2', href: '#' }, { text: '3', href: '#', selected: true }, { text: '4', href: '#' }, { text: '5', href: '#' }] })}}
{{hmctsPagination({ results: { from: 201, to: 250, count: 250 }, previous: { text: 'Previous', href: '#' }, items: [{ text: '1', href: '#' }, { text: '2', href: '#' }, { text: '3', href: '#' }, { text: '4', href: '#' }, { text: '5', href: '#', selected: true }] })}}

If there are more than 5 pages

  • Always show the first page number
  • Always show the last page number
  • Always show the current page number
  • Show the previous page number if applicable
  • Show the next page number if applicable
  • Always show five pages, show multiple next or previous pages if needed (see the first and last example)
  • Only show dots between numbers if they are non-successive
{{hmctsPagination({ results: { from: 1, to: 50, count: 500 }, next: { text: 'Next', href: '#' }, items: [{ text: '1', href: '#', selected: true }, { text: '2', href: '#' }, { text: '3', href: '#' }, { text: '4', href: '#' }, { type: 'dots' }, { text: '10', href: '#' }] })}}
{{hmctsPagination({ results: { from: 351, to: 400, count: 500 }, previous: { text: 'Previous', href: '#' }, next: { text: 'Next', href: '#' }, items: [{ text: '1', href: '#' }, { type: 'dots' }, { text: '6', href: '#' }, { text: '7', href: '#', selected: true }, { text: '8', href: '#' }, { type: 'dots' }, { text: '10', href: '#' }] })}}
{{hmctsPagination({ results: { from: 22151 | friendlyNumber, to: 22195 | friendlyNumber, count: 22195 | friendlyNumber }, previous: { text: 'Previous', href: '#' }, items: [{ text: '1', href: '#' }, { type: 'dots' }, { text: '437', href: '#' }, { text: '438', href: '#' }, { text: '439', href: '#' }, { text: '440', href: '#', selected: true }] })}}
